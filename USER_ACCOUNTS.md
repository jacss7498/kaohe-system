## 技术说明

### 数据库结构

users表包含以下字段：
- `id` - 用户ID
- `username` - 登录用户名（姓名）
- `password` - 密码哈希
- `name` - 显示姓名
- `role` - 角色（admin/leader/manager）
- `unit` - 所属单位
- `must_change_password` - 是否需要强制修改密码（1=是，0=否）
- `created_at` - 创建时间

### 查询所有用户

可以使用以下脚本查询所有用户信息：

```bash
# 进入容器
echo "Sun123456789" | sudo -S docker exec -it kaohe-backend sh

# 在容器内运行
cd /app
node -e "
const sqlite3 = require('sqlite3').verbose();
const db = new sqlite3.Database('/app/data/kaohe.db');
db.all('SELECT id, username, name, role, unit, must_change_password FROM users ORDER BY role, id', (err, rows) => {
  if (!err) console.table(rows);
  db.close();
});
"
```

## 密码重置

如果用户忘记密码，管理员可以运行以下脚本重置：

```bash
# 将用户密码重置为 123456 并要求强制修改
echo "Sun123456789" | sudo -S docker exec kaohe-backend node -e "
const sqlite3 = require('sqlite3').verbose();
const bcrypt = require('bcryptjs');
const db = new sqlite3.Database('/app/data/kaohe.db');
const password = bcrypt.hashSync('123456', 10);
db.run('UPDATE users SET password = ?, must_change_password = 1 WHERE username = ?',
  [password, '韩冬'], // 替换为要重置的用户名
  (err) => {
    if (err) console.error('重置失败:', err);
    else console.log('密码已重置为 123456');
    db.close();
  }
);
"
```
