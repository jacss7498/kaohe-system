# 单位科室及中队互评评分系统

## 项目简介

基于Web的移动端适配评分系统，实现本单位内部对6个科室及14个中队的线上考核互评。

## 技术栈

- **前端**: React + TypeScript + Vite + Tailwind CSS
- **后端**: Node.js + Express + TypeScript
- **数据库**: SQLite

## 快速开始

### 安装依赖

```bash
npm run install:all
```

### 开发模式

启动后端服务（端口3001）：
```bash
npm run dev:backend
```

启动前端服务（端口5173）：
```bash
npm run dev:frontend
```

### 生产构建

```bash
npm run build:backend
npm run build:frontend
```

## 用户角色

1. **超级管理员**: 系统初始化、任务管理、数据统计 
2. **领导班子成员** (5人): 可为所有科室和中队评分
3. **科室/中队负责人** (29人): 可为所有科室和中队互评

## 功能特性

- ✅ 用户登录认证
- ✅ 科室评分（6个科室，名额限制：优秀1个，良好2个）
- ✅ 中队评分（14个中队，名额限制：优秀2个，良好5个）
- ✅ 评分规则强制校验
- ✅ 电子签名功能
- ✅ 分数自动计算与排名
- ✅ 移动端适配
- ✅ 管理后台
